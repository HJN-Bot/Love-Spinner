# Love Wheel 互动模式（你描述的三栏作答 + 协商）

## 目标
一题一题聊清楚，不尴尬、不审讯：**先各自选择 → 再聊 → 最后协商一个“我们版答案”**。

---

## 页面布局（核心）

### 上半区：题目卡（居中）
- 标题：主题（如「隐私与边界」）
- 题干：问题正文
- 选项：A/B/C/D（最多 4 个；如需更多可变成 A-E 或做“下一页选项”）

### 中半区：双人作答区（左右对称）
- 左侧：男生（或「我」）作答
- 右侧：女生（或「TA」）作答
- 每侧都是 A/B/C/D 四个按钮（单选）
- **盲选模式（推荐默认）**：双方都选完才显示对方选项，避免迎合

### 下半区：聊一聊（Discussion）
- 展示 1-2 条追问（followUps），引导把抽象聊具体
- 一个简短输入框：记录“我们聊到的关键点”（可选）
- 可选：30/60/90 秒计时器（轻量游戏化）

### 底部：协商答案（Negotiated Answer）
- 显示：
  - 男生答案：A
  - 女生答案：C
  - 我们的答案（协商后）：B
- 「我们的答案」也用 A/B/C/D 选择按钮（单选）
- 右侧一个按钮：✅ 记录并进入下一题

---

## 数据结构（建议）

每道题记录三份答案 + 一段摘要：

```json
{
  "qid": "LWM-03-010",
  "male": "B",
  "female": "D",
  "negotiated": "C",
  "notes": "我们约定：冷静期要说清楚多久，最晚24h回沟通",
  "ts": 1700000000
}
```

---

## 交互细节（防尴尬 + 提体验）
- 当双方答案差异大：弹出「认真聊聊」提示（不评判），并优先展示追问
- `flags` 包含 redflag/boundary/money/family 的题：
  - 记录时提示“建议留一句‘可观察规则’”（例如：冷静期最长24h）
- 支持“跳过”：但会进入「未聊清清单」，方便回访
